40  // Initialize: Z() Clears mask register and Test Mode at S2 time and **sets module into Local mode**

44  // N-F(1)-A(O): Read threshold setting.
45  // N-F(O)-A(O): Read mask register. Q response in Remote mode.
46  // N-F(16)-A(O): Writes mask register pattern (W1-W16). X + Q response in Remote mode.
47  // N-F(17)-A(O): Writes threshold setting register; 11 bits, 10 bits of data (W1-W10)
48  // providing 1 mV resolution are used to program common threshold and 1 bit (W11)
49  // to set the manual threshold value (front-panel screwdriver control) to the
50  // threshold register. In this case, the data presented on W1-W10 is ignored.
51  // Q response in Remote mode.
52  // N-F(25)-A(O): Test function. Only channels not masked off by F(16) are triggered by
53  // internally generated 3 nsec wide pulse. Q response in Remote mode.
54  // N-F(24)-A(O): Set Local mode.
55  // N-F(26)-A(O): Set Remote mode.
56  //
57  // X: X=1 response is generated for every valid F,N,A.
58  // Q: Q=1 response is generated if the valid command can be executed.
59  //
